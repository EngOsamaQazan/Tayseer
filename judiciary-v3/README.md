# نظام إدارة القضايا - الإصدار 3.0

نظام متكامل لإدارة القضايا والشؤون القانونية، مبني كتطبيق مستقل باستخدام Node.js و React.

## المتطلبات

- **Node.js** v18 أو أعلى
- **MySQL** 5.7+ أو 8.0+ (قاعدة بيانات موجودة)
- **npm** v9 أو أعلى

## التثبيت

```bash
# 1. الانتقال لمجلد المشروع
cd judiciary-v3

# 2. تثبيت جميع التبعيات
npm run install:all

# 3. إعداد ملف البيئة
cp .env.example .env
# عدّل .env بإعدادات قاعدة البيانات
```

## إعداد قاعدة البيانات

عدّل ملف `.env` بإعدادات الاتصال:

```
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=tayseer
PORT=3001
```

## التشغيل

```bash
# تشغيل الخادم والواجهة معاً
npm run dev

# أو تشغيل كل جزء منفصلاً:
npm run server   # الخادم على http://localhost:3001
npm run client   # الواجهة على http://localhost:5173
```

## البنية التقنية

### الخادم (Backend)
- **Express.js** - إطار العمل
- **mysql2** - الاتصال بقاعدة البيانات
- **ExcelJS** - تصدير Excel
- **PDFMake** - تصدير PDF
- **Multer** - رفع الملفات

### الواجهة (Frontend)
- **React 18** - مكتبة الواجهة
- **Vite** - أداة البناء
- **Tailwind CSS** - التصميم
- **React Router** - التنقل
- **Heroicons** - الأيقونات

## الوحدات

| التبويب | الوصف |
|---------|-------|
| القضايا | عرض وإدارة جميع القضايا مع فلاتر متقدمة |
| الإجراءات | سجل إجراءات الأطراف مع الموافقة/الرفض |
| المثابرة | لوحة متابعة القضايا (أحمر/برتقالي/أخضر) |
| الشؤون القانونية | العقود في الحالة القانونية |
| التحصيل | إدارة عمليات التحصيل |

## الصفحات الإضافية

- **ملف القضية** - تفاصيل شاملة مع Pipeline و Timeline
- **إنشاء/تعديل قضية** - نموذج متكامل
- **إدخال مجمّع** - إنشاء قضايا دفعة واحدة
- **إدخال إجراءات مجمّع** - معالج 3 خطوات
- **لوحة المواعيد** - متابعة المواعيد النهائية
- **المخطط الزمني** - عرض زمني تسلسلي

## مسارات API

| المسار | الوصف |
|--------|-------|
| `/api/cases` | إدارة القضايا CRUD |
| `/api/actions` | إدارة الإجراءات CRUD |
| `/api/lookups/*` | بيانات القوائم المنسدلة |
| `/api/persistence` | تقارير المثابرة |
| `/api/legal` | الشؤون القانونية |
| `/api/collection` | التحصيل |
| `/api/deadlines` | المواعيد النهائية |
| `/api/timeline/case/:id` | المخطط الزمني |
| `/api/stats` | إحصائيات لوحة المعلومات |
| `/api/exports/*` | التصدير (Excel/PDF) |
| `/api/batch/*` | العمليات المجمّعة |
